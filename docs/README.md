# Search Algorithm Engine UI (Static Frontend)

This is a static React + Tailwind CSS frontend for the Search Algorithm Engine, designed for deployment on GitHub Pages.

## Features
- Clean, minimalist UI for search queries and results
- Responsive design for desktop and mobile
- Visual feedback (loading, no results, errors)
- Result ranking and highlighting (to be enhanced)
- Ready for GitHub Pages deployment

## Development

### Install dependencies
```
npm install
```

### Start dev server
```
npm start
```

### Build for production
```
npm run build
```

The production build will be in `docs/dist/`.

---

## Deployment
- Place all static files in the `docs/` folder for GitHub Pages.
- Use the provided GitHub Actions workflow for automatic deployment.
