Search Algorithm Engine

A small, handwritten text search engine with phrase queries and boolean operators.

View Repository CI Status

Try it locally

pip install -r requirements.txt
python -m src.cli --data data/raw --query "\"fast api\" AND database"

Query examples

  • "fast api" (phrase query)
  • fast AND api
  • fast OR api AND NOT database (NOT > AND > OR)
  • fast api (implicit AND)

GitHub Pages is currently served from /docs on the main branch. The React UI build can be deployed separately via the gh-pages branch workflow.
